MILLZMALEFICARUM CODEX v0.1
===========================

This folder contains all the project files for the MillzMaleficarum Codex project.

To work with this project:

1. Copy these files to your development environment
2. Install dependencies: npm install
3. Start the server: npm start

Key project structure:
- server.js - Main Express server
- public/ - Frontend files (HTML, CSS, JS)
- data/ - JSON content files

The project is designed to be a dynamic, JSON-driven digital magazine with a 
Vaporwave/Cryptic/Sacred Tech aesthetic.
